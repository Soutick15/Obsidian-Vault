
# Common Math Methods:

```Javascript

Math.abs(x): Returns the absolute value of x

Example: Math.abs(-7) // Output: 7

Math.ceil(x): Rounds x up to the nearest integer

Example: Math.ceil(4.2) // Output: 5

Math.floor(x): Rounds x down to the nearest integer

Example: Math.floor(4.7) // Output: 4

Math.round(x): Rounds x to the nearest integer

Example: Math.round(4.5) // Output: 5

Math.max(a, b, …): Returns the largest value from given numbers

Example: Math.max(3, 6, 2, 8) // Output: 8

Math.min(a, b, …): Returns the smallest value from given numbers

Example: Math.min(3, 6, 2, 8) // Output: 2

Math.pow(x, y): Returns x raised to the power of y (x^y)

Example: Math.pow(3, 2) // Output: 9

Math.sqrt(x) // Returns the square root of x

Example: Math.sqrt(16) // Output: 4

Math.random(): // Returns a random number between 0 (inclusive) and 1 (exclusive)

Example: Math.random() // Output: (random number like 0.5342)

Math.trunc(x) // Removes the decimal part of x, returning only the integer part

Example: Math.trunc(5.9) // Output: 5

Math.sign(x) // Returns the sign of x: -1 (negative), 0, or 1 (positive)

Example: Math.sign(-10) // Output: -1

Math.log(x) // Returns the natural logarithm of x (base e)

Example: Math.log(10) // Output: 2.3025 (approx.)

Math.exp(x) // Returns e raised to the power of x (e^x)

Example: Math.exp(1) // Output: 2.718 (approx., value of e)

Math.sin(x) // Returns the sine of x (in radians)

Example: Math.sin(Math.PI / 2) // Output: 1

Math.cos(x) // Returns the cosine of x (in radians)

Example: Math.cos(0) // Output: 1

Math.tan(x) // Returns the tangent of x (in radians)

Example: Math.tan(Math.PI / 4) // Output: 1

Math.PI // Returns the constant value of PI

Example: Math.PI // Output: 3.14159 (approx.)

Math.E // Returns Euler's number (approx. 2.718)

Example: Math.E // Output: 2.718
```
