# ECMAScript 6 (ES6)

> [!NOTE]
> **ECMAScript 2015 (ES6)** is a major update to JavaScript that introduced modern syntax and features, making code more readable, maintainable, and easier to write.

---

## Why ES6?

Before ES6, JavaScript code was often verbose and lacked many modern programming features. ES6 introduced improvements for:

- Cleaner syntax
- Better variable declarations
- Improved object-oriented programming
- Asynchronous programming
- Better handling of arrays and objects
- Modular development

---

# Major ES6 Features

| Feature | Purpose |
|---------|---------|
| **Arrow Functions (`=>`)** | Shorter syntax for writing functions and lexical `this` binding. |
| **Classes** | Cleaner syntax for object creation and inheritance. |
| **Template Literals** | Supports string interpolation and multi-line strings using backticks (`` ` ``). |
| **Destructuring Assignment** | Extract values from arrays and objects into variables. |
| **Default Parameters** | Assign default values to function parameters. |
| **Rest Operator (`...`)** | Collect multiple function arguments into an array. |
| **Spread Operator (`...`)** | Expand arrays or objects into individual elements. |
| **Promises** | Handle asynchronous operations without callback nesting. |
| **`let` & `const`** | Block-scoped variable declarations. |
| **Iterators & Generators** | Create custom iterable sequences and pause/resume function execution. |
| **Modules (`import` / `export`)** | Split applications into reusable modules. |
| **Map & Set** | New collection data structures for storing unique values and key-value pairs. |
| **Symbols** | Create unique and immutable identifiers. |

## Real Project Usage

| Feature | Common Usage |
|---------|--------------|
| Arrow Functions | React components, callbacks, array methods |
| Template Literals | Dynamic HTML, API URLs, logging |
| Destructuring | API responses, React props, function parameters |
| Rest & Spread | Copying arrays/objects, merging data |
| Promises | API calls (`fetch`, Axios) |
| Modules | Organizing large applications |
| Classes | Object-oriented programming |
| Map & Set | Caching, unique values, lookup tables |
